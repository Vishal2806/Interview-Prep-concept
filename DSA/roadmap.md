# 📚 Data Structures & Algorithms (DSA) Roadmap

## 🔹 Types of Data Structures

### 1. Linear Data Structures

* Arrays
* Linked List
* Stack
* Queue
* Hash Tables

### 2. Non-Linear Data Structures

* Trees
* Binary Search Tree (BST)
* Heaps
* Graphs
* Trie
* Union Find (Disjoint Set)

---

## 🔹 Algorithms

### Sorting Algorithms

* Bubble Sort
* Selection Sort
* Insertion Sort
* Merge Sort
* Quick Sort

### Searching Algorithms

* Binary Search

### Bit Manipulation

* AND, OR, XOR
* Bit Masking

---

## 🔹 Tree Traversal Algorithms

* Inorder
* Preorder
* Postorder
* Level Order (BFS)

---

## 🔹 Graph Algorithms

* DFS (Depth First Search)
* BFS (Breadth First Search)
* Topological Sort

### Shortest Path Algorithms

* Dijkstra
* Bellman-Ford

---

## 🔹 Problem Solving Techniques (Patterns)

* Two Pointers
* Sliding Window
* Prefix Sum
* Fast & Slow Pointer
* Divide & Conquer
* Greedy
* Recursion
* Backtracking
* Dynamic Programming
* Top K Elements

---

## 🔹 Complexity Analysis

* Time Complexity (Big-O)
* Space Complexity

👉 Always analyze your solution before and after coding.

---

## 🚀 Order to Learn DSA

1. Arrays
2. Linked Lists
3. Stacks & Queues
4. Hash Tables
5. Two Pointers / Sliding Window
6. Recursion / Backtracking
7. Sorting
8. Searching
9. Bit Manipulation
10. Trees / BST / Trie
11. Heaps
12. Greedy
13. Dynamic Programming
14. Graphs / Union Find

---

## 💡 Tips for Learning

* Think in **patterns**, not just problems
* Practice daily (even 2–3 problems is enough)
* Revise regularly
* Focus on understanding, not memorizing
* Solve problems from different platforms

---

## 🔁 Golden Rule

> **Repetition + Revision = Mastery**
